# for example -
# 1. First move to the directory Assignment_Word_Stemming
# 2. Compile the code by  writing on command - "javac MainClass.java"
# 3. Run the command "java MainClass <location_file_content>"
# 4. It will open GUI and when you click the button it will show you stemmed words of content of the file.
# Bydefault you can run "java MainClass /content.txt " after compilation it will show the content of the file "content.txt".
